# KQ Invoice Generator

A professional invoice generator built with HTML, CSS, and JavaScript. Create, preview, and download beautifully designed invoices as PDF - entirely client-side.

## Features

- **Invoice Form** - Enter client details, services/items, amounts, and payment info
- **Live Preview** - Realistic invoice preview with company branding
- **PDF Export** - Download invoices as A4 PDF via html2pdf.js
- **Payment Tracking** - Track total deal, amount received, and balance due
- **Status Badges** - Mark invoices as Paid, Unpaid, or Partially Paid
- **Fully Offline** - All processing happens in the browser; no server needed

## Usage

Open \KQ_Invoice_Generator.html\ in any modern browser, fill out the form, and click **Preview Invoice**, then **Download PDF**.

## Tech Stack

- Vanilla HTML/CSS/JS
- [html2pdf.js](https://github.com/eKoopmans/html2pdf.js) - Client-side PDF generation
- Google Fonts (Playfair Display, DM Sans)
